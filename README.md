# yongcat

한국어 용언(동사/형용사)의 활용형을 생성하는 Rust 라이브러리입니다.

## 특징

- 표준국어대사전 기반으로 용언 1,721개 지원으로 시작
- 라이브러리가 안정된 다음 용언을 추가할 계획
- 어미 그룹별 활용형 자동 생성 (아/어 계열 모음조화 처리)
- 동음이의어 구분 (`dict_id`)
- 등급별 컴파일 (학습 등급 A/B/C 필터링)

## 현황

```rust
let yongeons = yongcat::load_yongeons();
let eomis = yongcat::load_eomis();
println!("용언: {}개", yongeons.len());  // 1721
println!("어미: {}개", eomis.len());     // 26
```

## 사용법

```rust
use yongcat::eomi::ah_eo;

let yongeons = yongcat::load_yongeons();

// 단어 문자열로 활용형 생성 (동음이의어 전체 처리)
let results = yongcat::postfix(&yongeons, "가다", &ah_eo::AYO);
for (yongeon, conjugated) in &results {
    println!("{} → {}", yongeon.base_form, conjugated);
    // 가다 → 가요
}

// 단일 용언으로 활용형 생성
let meok = &yongcat::find_yongeon(&yongeons, "먹다")[0];
let result = yongcat::postfix_word(meok, &ah_eo::ASS);
// "먹었"
```

## 어미 (26개)

`load_eomis()`로 전체 어미 목록을 로드하고, `find_eomi()`로 문자열 검색할 수 있습니다.

### ah_eo (아/어 계열, 9개)

| 상수 | 어미 | 용도 |
|------|------|------|
| `A` | 아/어 | 종결 (반말) |
| `AYO` | 아요/어요 | 종결 (해요체) |
| `ASEO` | 아서/어서 | 연결 (이유) |
| `ADO` | 아도/어도 | 연결 (양보) |
| `AYA` | 아야/어야 | 연결 (조건) |
| `ARA` | 아라/어라 | 명령 |
| `ASS` | 았/었 | 과거 시제 |
| `ASS_EOYO` | 았어요/었어요 | 과거 해요체 |
| `ASS_SEUMNIDA` | 았습니다/었습니다 | 과거 합쇼체 |

### 규칙 활용

| 규칙 | 조건 | 예시 | 상태 |
|------|------|------|------|
| 모음조화 | 양성모음(ㅏ, ㅗ) → 아, 음성모음 → 어 | 가+아요 → 가요, 먹+어요 → 먹어요 | 구현 완료 |
| ㅏ+ㅏ 탈락 | 어간 ㅏ + 어미 ㅏ | 가+아요 → 가요 | 구현 완료 |
| ㅓ+ㅓ 탈락 | 어간 ㅓ + 어미 ㅓ | 서+어요 → 서요 | 구현 완료 |
| ㅗ+ㅏ 축약 | 어간 ㅗ + 어미 ㅏ → ㅘ | 오+아요 → 와요 | 구현 완료 |
| ㅜ+ㅓ 축약 | 어간 ㅜ + 어미 ㅓ → ㅝ | 주+어요 → 줘요 | 구현 완료 |

| ㅡ 탈락 | 어간 ㅡ + 어미 ㅓ → ㅓ | 크+어요 → 커요 | 구현 완료 |

## 모듈 구조

| 모듈 | 역할 |
|------|------|
| `eomi` | 어미 데이터 및 `Eomi` 열거형 (`ah_eo`, `fixed`, `plain`) |
| `irregular` | 불규칙 활용 유형별 처리 (여/ㄷ/ㅂ/ㅅ/ㄹ/우/러/으) |
| `join` | 어간-어미 접합 디스패처 (irregular → regular) |
| `merge` | 음운 축약/탈락 디스패처 (irregular → regular) |
| `syllable` | 한글 음절 분해/합성 |
| `yongeon` | `Yongeon` 구조체, `moeum_joha()` 모음조화 |
| `types` | `YongeonType`, `IrregularType` 열거형 |

## 등급별 컴파일

Cargo feature로 특정 학습 등급의 용언만 포함할 수 있습니다.

```bash
cargo build --features grade-a            # A등급만 (230개)
cargo build --features "grade-a,grade-b"  # A+B등급 (863개)
cargo build                               # 전체 (1,721개)
```

## 한계

### 품사별 어미 제한 없음

동사에만 쓸 수 있는 어미(예: "는"은 동사 현재형)를 형용사에 적용해도 오류 없이 활용형이 생성됩니다. 어미 선택 제한은 현재 구현하지 않습니다.

### 동음이의어 구분

1,721개 용언 중 69개 기본형(156개 용언)이 동음이의어입니다. 대부분은 품사만 다르고 활용형이 같지만, 불규칙 유형이 달라 활용형이 달라지는 경우가 있습니다.

| 기본형 | 차이 | 예시 |
|--------|------|------|
| 걷다 | ㄷ불규칙 vs 규칙 | 걸어요 (걷다02) vs 걷어요 (걷다03) |
| 묻다 | ㄷ불규칙 vs 규칙 | 물어요 (묻다03) vs 묻어요 (묻다01) |
| 이르다 | 러불규칙 vs 르불규칙 | 이르러요 (이르다01) vs 일라요 (이르다03) |

이 경우 `find_yongeon`이 동음이의어를 모두 반환하므로, `dict_id`로 구별해야 합니다.

### 미구현 불규칙

ㅎ, 르 불규칙은 아직 구현되지 않았습니다. 해당 용언에 아/어 계열 어미를 적용하면 잘못된 활용형이 생성됩니다.
